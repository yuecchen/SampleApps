GET Methods

The following API's will pull randomly generated Photo Album data
  • Mini: Photo album data pull [~22K in size] {{ getmini }}
  • Small: Photo album data pull [~220K in size] {{ getsmall }}
  • Medium: Photo album data pull [~1.1M in size] {{ getmedium }}
  • Large: Photo album data pull [~2.2M in size] {{ getlarge }}
  • XL: Photo album data pull [~22M in size] {{ getxl }}
  • XXL: Photo album data pull [~110M in size] {{ getxxl }}
  • XXXL: Photo album data pull [~220M in size] {{ getxxxl }}

PUT Method

The following API is a simulated PUT. No actual data will be updated. See BlueMix log for evidence this took place Note that the PUT *must* be accompanied with the JSON header and data
  • PUT Test {{ putsim }}
Example Curl Command:
curl -H "Content-Type: application/json" -X PUT -d '{"title":"Updated by BlueMix Availability Monitoring"}' http://localhost:8080/method/api/put/sim

PUT Method with Security

The following API is a simulated PUT with security. No actual data will be updated. See BlueMix log for evidence this took place Note that the PUT *must* be accompanied with the JSON header, data and valid credentials.
  • PUT Test {{ putsimsecured }}
Example Curl Command:
